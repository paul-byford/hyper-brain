# Writing good notes

A good note is easy to find later and easy to connect to what you already know.
This page is part of getting started; see [[Getting started with the brain]] first
if you have not.

## Give it a clear title

The title is what search and other notes match on, so make it specific. "Q3 fraud
model rollout" beats "notes". A precise title is also what lets other notes link to
this one.

## Connect notes with wikilinks

Notes are connected with **wikilinks**: write the title of another note in double
square brackets and the two notes become linked in the graph. For example, writing
`[[Getting started with the brain]]` in a note links it to that page.

Two rules worth knowing:

- **Links are explicit.** A note only connects to notes you actually reference with
  `[[...]]`. Plain prose with no wikilinks stays unconnected, which is why a fresh
  set of notes shows no edges until you add links.
- **Links stay inside a space.** A wikilink only resolves to another document in the
  same domain. Inside your personal space, your notes link to each other; a link
  that points outside your space is dropped, so linking can never surface something
  you are not allowed to see. This mirrors how the [[Glossary]] defines a domain.

## Tag for grouping

Add a few tags so related notes cluster together even before you link them.
